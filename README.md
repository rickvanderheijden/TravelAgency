Quick Build
-------
If you want to bootstrap Maven, you'll need:
- Java 1.7+
- Maven 3.0.5 or later

- Build project local
```
mvn -Dmaven.test.skip=true package
```

- Run project local
```
mvn spring-boot:run
```

Install docker-compose
- https://docs.docker.com/compose/install/

- Run docker-compose
```
docker-compose up
```
